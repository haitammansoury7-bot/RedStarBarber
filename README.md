Étapes pour finaliser et publier
1. Vérifier `index.html` :
   - Le numéro WhatsApp publié est `+375256230705` (variable `ownerPhone` dans le script). Modifiez si nécessaire.
   - Remplacez l'URL canonique si vous utilisez un domaine personnalisé.

2. Mettre à jour `sitemap.xml` et `robots.txt` si vous changez l'URL publique.

3. Publier sur GitHub Pages :
   - Les fichiers sont poussés sur la branche `main`.
   - Dans GitHub: Settings → Pages → sélectionnez `main` branch, dossier `/ (root)` puis Enregistrer.

4. Soumettre à Google :
   - Dans Google Search Console, ajoutez la propriété (domaine ou URL) puis soumettez le sitemap : `https://haitammansoury7-bot.github.io/RedStarBarber/sitemap.xml`.

Conseils SEO rapides
- Ajoutez du contenu unique et des pages supplémentaires (Ex: page "À propos", "Galerie", "Avis clients").
- Ajoutez des balises alt aux images et une image OG (og:image) de bonne taille (1200x630).
- Obtenez quelques liens locaux (annuaire, réseaux sociaux) pour aider l'indexation locale.
- Publiez régulièrement et mettez à jour `sitemap.xml` si vous ajoutez des pages.
